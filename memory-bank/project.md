# Project Overview

The Photo App is a mobile application (React Native/Expo) with a FastAPI backend designed to catalog and manage items using AI-powered image analysis.

## Architecture
- **Frontend**: React Native with Expo.
- **Backend**: FastAPI (Python).
- **Database**: Supabase (PostgreSQL).
- **AI**: Google Gemini API for image analysis.
- **Storage**: Supabase Storage for item images.

## Key Features
- AI-driven item analysis and categorization.
- Inventory management across different rooms.
- Multilingual support (English/German).
- Favoriting and searching capabilities.

## Technical Stack
- Supabase for Auth, DB, and Storage.
- Postgrest for data access.
- React Navigation for app flow.
- Reanimated for smooth UI transitions.
